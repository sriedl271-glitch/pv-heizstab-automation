# PV Heizstab Automation

Dieses Projekt ist die Basis für eine cloudbasierte PV-Heizstab-Automation.

## Ziel
- PV-Daten aus iSolarCloud lesen
- Regeln auswerten
- Pushover und E-Mail senden
- später optional TYDOM ansteuern

## Phase
Priorität 1:
- Grundsystem
- Testnachrichten
- Logik mit Testdaten
